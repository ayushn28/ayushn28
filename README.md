### 🚀 Featured Projects

#### 🛡️ Sentinel IDS — Network Intrusion Detection System &nbsp; [Code](https://github.com/ayushn28/nids-intrusion-detection-system) · [Live Demo](https://nids-1-v8dl.onrender.com/)

> Real-time network security platform that detects & classifies malicious traffic into **14 attack categories**

- Trained a **Voting Ensemble (RF + LR + Extra Trees)** on **2.8M+ network flows** from CIC-IDS 2017
- Achieved **96.85% accuracy** and **0.994 AUC-ROC** with 5-fold cross-validation
- **Reduced false positives by 40%** via threshold tuning on severely imbalanced data (83/17 split)
- FastAPI backend classifies **500K+ flows in under 2 seconds**; SHAP per-row explainability panels
- WebSocket real-time monitoring; deployed on **Vercel + Render** with CI/CD via GitHub

`Python` `FastAPI` `React` `scikit-learn` `SHAP` `WebSockets` `Docker` `Vercel` `Render`
